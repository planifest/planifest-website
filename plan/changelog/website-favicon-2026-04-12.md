# Changelog: Branded Favicons
Date: 2026-04-12
Feature: website-favicon

## Summary
Integrated a complete set of branded favicons and a webmanifest into the Planifest website to ensure consistent visual identity across browser tabs, shortcuts, and mobile devices.

## Changes
- **Assets**: Added `apple-touch-icon.png`, `favicon-32x32.png`, `favicon-16x16.png`, and `favicon.ico` to the web app assets.
- **Webmanifest**: Added `site.webmanifest` to support Android chrome and PWA features.
- **HTML**: Updated `index.html` head to link all favicon assets.
- **Branding**: Added source brand assets to the `graphics/` directory.

## Impact
- **Visibility**: Improved bookmark and tab visibility.
- **Branding**: Uniform look across all deployment environments.
