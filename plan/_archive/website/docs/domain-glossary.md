# Domain Glossary - Planifest Website

> The ubiquitous language used within the feature.

| Term | Definition |
|---|---|
| **Feature** | A discrete project or block of work managed by Planifest. Here, "website" is the feature. |
| **Hero Section** | The prominent top section of a website's landing page containing a headline, sub-headline, visuals, and clear call-to-action buttons. |
| **Doc-renderer** | The build-time mechanism (within Vite) responsible for transforming source Markdown documents from `planifest-docs` into static HTML. |
| **Metadata Stripping** | The process of parsing the markdown to specifically identify and remove specific Markdown frontmatter, version strings, and "Version Log" tables prior to generation. |
| **Planifest Framework** | The primary subject matter of the site; the AI agentic framework that this landing page exists to promote and explain. |
| **Vite** | The frontend build tool chosen as the generator for this static site infrastructure. |
| **Glassmorphism** | A UI design trend featuring a frosted-glass effect, background blur, and semi-transparent layers to be used in styling the website. |
